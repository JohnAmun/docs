<Info>
  An auto-swap rule has a **minimum USD threshold** — the swap only triggers once a replenished wallet crosses it. It uses the same swap engine, so the same cost applies: **miner fee \+ 0.5% \+ provider spread**.
</Info>

<Warning>
  Auto-swap requires **autosign to be ON**. If autosign is disabled, auto-swap stops working.
</Warning>
